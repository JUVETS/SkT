# 1. Projektarbeit "Skriptingtechnik PowerShell"

## 1.1. Organisation

|                     |                                                                    |
| :------------------ | :----------------------------------------------------------------- |
| **Lernziele**       | PowerShell Techniken und Einsatzbereiche untersuchen und einsetzen |
| **Sozialform**      | Gruppenarbeit (max. 3 Mitglieder)                                  |
| **Auftrag**         | siehe unten                                                        |
| **Hilfsmittel**     | Internet                                                           |
| **Zeitbedarf**      | Total ca. 3h / pro Mitglied                                        |
| **Lösungselemente** | Vollständiges Projekt inkl. Dokumentation u. Präsentation          |

</br>

## 1.2. Ziel

Die Studierenden gewinnen eine Einsicht über die verschiedenen Powershell Techniken. Sie verstehen die verschiedenen Einsatzgebiete und können die Techniken praktisch anwenden.

---

</br>

## 1.3. Aufträge

Ermitteln Sie alle wichtigen Informationen über das Ihnen zugeteilte Thema und erstellen Sie eine kleine Zusammenfassung inkl. Lösungsbeispiel. Dabei sollen folgende Punkte untersucht werden:

- Funktionsumfang / Merkmale
- Einsatzbereich (Beispiele)
- Praktisches Beispiel (Live Demo)

Stellen Sie Ihre Ergebnisse mittels einer Kurzpräsentation der Klasse vor. Verwenden Sie dabei die Hilfsmittel wie Flow-Charts, Beamer, Wandtafel usw. und verweisen Sie ggf. auf weitere die Literatur. Die Zusammenfassungen und Beispiel sind dann den anderen Klassenkameraden zur Verfügung zu stellen.

> **Dauer der Kurzpräsentation: ca. 15min**

---

</br>

## 1.4. Gruppeneinteilung

In einzelnen Teams sollen folgende Powershell Techniken untersucht werden:

### 1.4.1. PowerShell Remoting

- Einrichten und Rechte
- Fernausführungs-Commandlets
- Fernausführung von Skripten

### 1.4.2. Verwendung von .NET-Klassen und COM-Klassen

- Erzeugen von Instanzen
- Generische Klassen nutzen
- Zugriff auf bestehende Objekte
- Automatisierungen

### 1.4.3. Active Directory

- Benutzer- und Gruppenverwaltung im Active Directory
- Suche im Active Directory
- System.DirectoryServices
- Verwaltung der Organisationseinheiten

### 1.4.4. Entwicklung eigener Commandlets mit C#

- Grundkonzept
- Erstellung eines Commandlets mit Rückgabeobjekten
- Verarbeitung von Pipeline-Eingaben
- Integration von Hilfeinformationen
  
### 1.4.5. Datenbanken und grafische Benutzeroberflächen

- ADO.NET Grundlagen
- Datenzugriff mit Powershell
- Eingabemasken / WPF Powershell Kit (WPK)
